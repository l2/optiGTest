0001 optiGTest 
0002 =======
0003 
0004 [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.1248985.svg)](https://doi.org/10.5281/zenodo.1248985)
0005 
0006 optiGTest is a MATLAB's toolbox which regroups many test functions used for studying the performance of approximation techniques and optimization strategy. In particular, gradient sof the test functions are provided.
0007 
0008 Features
0009 ------
0010 optiGTest is able to 
0011 
0012 * Provide derivatives of the test functions
0013 * Dealing with any dimension chosen by the user
0014 * Provide global minimum (value and associated set of parameters)
0015

0017 ------
0018 * unConstrained problems: `Example_unConstrained.m`

0130 Dependency
0131 ------
0132 
0133 **optiGTest** uses the **gradFD** class for checking gradients computation. Loading this class is optional but some functionalities will not be available which is not very important if you only want to use the main purpose of this toolbox (computing responses of objective and constraints functions and their gradients).
0134 
0135 If you forget to use the `--recursive` option when you clone the repository, please use the command `submodule --init` in order to initialize the missing class **gradFD**.
